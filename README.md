# Todo - LiSSSS

A new Flutter project.

## Getting Started

This project is a starting point for a Flutter application.

A new version for to do list app "To Do List Sederhana Sangat Simple Sekali".

- Kategori bisa tak terbatas sampai list malah jadi chaos.

- UI yang plain bikin cepat bosan dan ditinggalkan.
